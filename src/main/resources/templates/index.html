Welcome to my IP data collection application!

A couple of things before we get started:

This is a project made by Nikita Ponomarev for the Server Programming course during Spring of 2020 taught by Jukka Juslin
The purpose of this project is to:

  • 1. Demonstrate the data that can be collected through users public ip-address just by visiting a website
  • 2. Use a semi-custom back-end that stores user credentials and data collected in a database that is cleared once in an hour
  • 3. Provide an API for registered users where they could use the data that was collected (not really recommended)
